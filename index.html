JSON-ANALYZER

Analyze and measure where you should spend your time optimizing your json files. Use it online, on the command line or your node/browser project.
json-analyzer path/to/file.json
const jsonAnalyzer = require("json-analyzer"); // or: import jsonAnalyzer from "json-analyzer"; const jsonObj = { "hello": "world" }; console.log( jsonAnalyzer({ json: jsonObj /* verbose: true/false, maxDepth: positive_number, target: "nodeName[positive_number].nodeName" */}) );
